# 🍽️ Restaurant Menu Website

The Rustic Spoon restaurant menu website built using 100% pure, semantic HTML5.

## Live preview:
Link: [The Rustic Spoon Restaurant](https://the-rustic-spoon.netlify.app)

---

## Features

* **100% Pure HTML:** Built completely without CSS or JavaScript, leveraging traditional structure techniques.
* **Semantic Layout:** Uses proper native landmarks (`<header>`, `<main>`, `<section>`, `<footer>`) to ensure great readability and SEO practices.
* **Smooth In-Page Navigation:** Quick-jump menu anchors (`#Appetizers`, `#MainCourses`, `#Desserts`) for a seamless user experience.
* **Organized Menu Matrix:** Displays real-world content structure using tables, consistent asset sizing, description typography, and dynamic promotional markdown (such as `<del>` and `<strong>` for live price cuts).
